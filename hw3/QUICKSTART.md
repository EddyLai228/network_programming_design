# Quick Start Guide

This guide will help you quickly set up and demo the Game Store System.

## Step 1: Start the Servers (5 minutes)

### Terminal 1: Start Servers
```bash
cd hw3/server
./start_servers.sh
```

You should see:
```
Starting Developer Server on port 8001...
Starting Lobby Server on port 8002...
Both servers are now running!
```

Keep this terminal running.
