# 貪食蛇 v1.1.0 - 增強版

## 遊戲簡介
經典的貪食蛇遊戲，現在具有更多功能和改進的視覺效果！

## 新功能 (v1.1.0)

### 🎮 難度選擇
- **簡單 (Easy)**: 較慢的速度，適合初學者
- **中等 (Medium)**: 標準速度，平衡的挑戰
- **困難 (Hard)**: 快速移動，高難度挑戰

### ⏸️ 暫停功能
- 按 **空白鍵** 隨時暫停或繼續遊戲
- 在暫停時畫面會顯示「暫停」文字

### 🏆 最高分記錄
- 系統會自動記錄您的最高分
- 最高分會保存在本地文件中
- 每次遊戲時都能看到您的歷史最佳成績

### 🎨 視覺改進
- 蛇頭使用亮綠色圓形，更容易識別方向
- 蛇身使用漸變效果，看起來更立體
- 食物使用紅色圓形，帶有邊框
- 背景添加了網格線
- 更好的配色方案

### 📋 遊戲說明
- 介面上顯示完整的控制說明
- 隨時知道如何操作遊戲

## 遊戲玩法

### 目標
- 控制蛇移動吃食物
- 每吃一個食物得 10 分
- 蛇會變長
- 不要撞到牆壁或自己

### 操作方式
```
方向鍵 (↑↓←→)  - 控制蛇的移動方向
空白鍵          - 暫停/繼續遊戲
R 鍵           - 重新開始遊戲
難度按鈕        - 切換遊戲難度
```

## 遊戲規則

1. **移動**: 蛇會持續朝當前方向移動
2. **吃食物**: 碰到紅色圓點即可得分並增長
3. **遊戲結束條件**:
   - 撞到牆壁
   - 撞到自己的身體
4. **得分**: 每個食物 10 分

## 提示與技巧

1. **選擇合適的難度**: 如果是第一次玩，建議從「簡單」開始
2. **計劃路線**: 不要只想著吃食物，也要考慮吃完後的位置
3. **利用空間**: 盡量在開闊的區域移動，避免把自己困住
4. **使用暫停**: 在複雜情況下，可以暫停思考下一步
5. **練習方向控制**: 熟悉方向鍵的操作，避免按錯導致遊戲結束

## 常見問題

**Q: 遊戲太快/太慢怎麼辦？**
A: 點擊底部的難度按鈕切換速度。

**Q: 最高分記錄會保存嗎？**
A: 會的，記錄保存在 `high_score.json` 文件中。

**Q: 可以在遊戲中改變難度嗎？**
A: 可以，但會重新開始當前遊戲。

**Q: 按鍵沒有反應？**
A: 確保遊戲視窗是當前焦點（點擊一下遊戲視窗）。
