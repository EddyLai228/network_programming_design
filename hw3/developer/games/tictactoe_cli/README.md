# TicTacToe CLI 對戰版

## 遊戲說明
這是一個雙人連線對戰的井字遊戲，使用命令列介面。兩位玩家透過網路連線進行即時對戰。

## 遊戲特色
- 🎮 雙人連線對戰
- 💻 簡潔的命令列介面
- 🌐 支援網路連線
- ⚡ 即時同步

## 遊戲規則
- 兩位玩家輪流在 3x3 棋盤上下棋
- 玩家 X 先手，玩家 O 後手
- 先連成一線（橫、直、斜）者獲勝
- 棋盤填滿且無人獲勝則為平局

## 如何遊玩
1. 房主會自動啟動遊戲伺服器
2. 所有玩家會自動連接到伺服器
3. 輪到你時，輸入座標 (行 列)，例如: `0 1`
4. 座標範圍: 0-2
5. 遊戲會自動判定勝負

## 技術規格
- 類型: CLI_MULTIPLAYER
- 玩家數: 2 人
- 版本: 1.0.0
