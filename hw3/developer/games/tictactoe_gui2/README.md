# TicTacToe GUI 對戰版 - 井字遊戲雙人對戰

## 遊戲簡介

井字遊戲（Tic-Tac-Toe）GUI 雙人對戰版，透過網路與其他玩家即時對戰。

## 遊戲規則

1. 3×3 的棋盤
2. 兩位玩家分別使用 X 和 O
3. 輪流在空白方格下棋
4. 先將三個符號連成一線（橫、豎、斜）者獲勝
5. 棋盤下滿無人獲勝則為平局

## 操作方式

### 玩家操作
- **滑鼠點擊**：輪到你時，點擊空白方格下棋
- **等待對手**：等待對手下棋時，按鈕會被鎖定

### 視覺提示
- **綠色文字**：輪到你了
- **橙色文字**：等待對手
- **紅色文字**：遊戲結束

## 遊戲流程

1. **等待連線**：進入遊戲後等待另一位玩家
2. **分配角色**：系統自動分配 X 或 O
3. **輪流下棋**：X 先手，O 後手
4. **即時同步**：每步棋都會立即同步到對手
5. **判定勝負**：自動判定勝負或平局

## 遊戲特色

- 🎨 美觀的 GUI 介面
- 🌐 網路即時對戰
- 🎮 簡單易用的操作
- 🔄 自動同步遊戲狀態
- 🏆 自動判定勝負
- 📝 自動記錄遊戲結果

## 技術實現

- 使用 Python Tkinter 實現 GUI
- Socket 網路通訊
- 多執行緒處理訊息接收
- JSON 格式傳輸遊戲狀態
- 自動寫入 game_result.txt

## 玩家角色

### 玩家 X (先手)
- 符號：X
- 顏色：紅色 (#e74c3c)
- 優勢：先手

### 玩家 O (後手)  
- 符號：O
- 顏色：藍色 (#3498db)
- 優勢：可以觀察對手策略

## 遊戲結果記錄

遊戲結束後會自動記錄：
- `玩家 X 獲勝!`
- `玩家 O 獲勝!`
- `平局!`

結果會寫入 `game_result.txt` 供系統追蹤。
