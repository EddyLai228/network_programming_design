# TicTacToe GUI - 井字遊戲 GUI 版

## 遊戲簡介

經典的井字遊戲（Tic-Tac-Toe）GUI 版本，單人對戰電腦 AI。

## 遊戲規則

1. 3×3 的棋盤
2. 玩家使用 X，電腦使用 O
3. 玩家先手
4. 點擊空白方格下棋
5. 先將三個符號連成一線（橫、豎、斜）者獲勝
6. 棋盤下滿無人獲勝則為平局

## 操作方式

- **滑鼠點擊**：點擊空白方格下棋
- **開始遊戲**：點擊「開始遊戲」按鈕
- **重新開始**：點擊「重新開始」按鈕

## AI 策略

電腦 AI 使用以下策略：
1. 如果能獲勝，立即走該步
2. 如果玩家下一步能獲勝，阻擋之
3. 優先佔據中心位置
4. 其次佔據角落位置
5. 最後佔據任意空位

## 遊戲特色

- 🎨 美觀的 GUI 界面
- 🤖 智能 AI 對手
- 🎮 簡單易用的操作
- 🏆 自動判定勝負
- 📝 自動記錄遊戲結果

## 技術實現

- 使用 Python Tkinter 實現 GUI
- 單人遊戲模式
- 自動寫入 game_result.txt 供系統記錄
