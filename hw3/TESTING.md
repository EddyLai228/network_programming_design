# Testing Checklist

在正式 Demo 前，請使用此檢查清單測試所有功能。

## 環境測試

### 伺服器環境
- [ ] Python 版本 >= 3.7
- [ ] 所有腳本有執行權限
- [ ] Port 8001 和 8002 未被佔用
- [ ] 網路連線正常

### 客戶端環境
- [ ] Python 版本 >= 3.7
- [ ] 可以連線到伺服器
- [ ] tkinter 可用 (GUI 遊戲需要)
